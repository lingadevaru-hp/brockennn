# Portfolio — lingadevaru.in

## Summary

A custom single-file HTML portfolio site. Live at [lingadevaru.in](https://lingadevaru.in). Designed around a bioluminescent forest-ocean aesthetic — dark, glowing, and animated.

## Theme

**Bioluminescent forest-ocean** — deep dark background, glowing particles, organic glowing plant forms, and ocean-depth atmosphere. Not a template — built custom.

## Features

| Feature | Detail |
|---------|--------|
| **Boot screen** | Animated loader on first visit |
| **Background** | Animated network graph with glowing nodes |
| **Particles** | Floating bioluminescent particles |
| **Plants** | SVG animated glowing plant forms |
| **Parallax** | Depth scrolling on key sections |
| **Video backgrounds** | HLS video streams for section backgrounds |
| **Cursor** | Custom cursor with trail effect |
| **Cards** | 3D tilt effect on hover |
| **Glassmorphism** | Liquid glass morphism UI elements |
| **Font** | Space Grotesk |
