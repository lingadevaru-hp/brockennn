# SearchXEng

## Summary

A Dockerized fork/rebrand of [SearXNG](https://github.com/searxng/searxng) — a privacy-focused meta-search engine that queries multiple search engines simultaneously and aggregates results without tracking users.

**Status: Running locally only. Not publicly hosted.**

## What It Is

SearXNG is a well-known open-source meta-search engine. SearchXEng is a personal rebrand of it with a custom name. It is self-hosted via Docker on a local Linux machine — not accessible from the internet.

## Current Status

- **Running:** Locally on personal machine
- **Platform:** Started on Linux Mint, now running on Omarchy (current Linux setup)
- **Not hosted:** There is no public URL. Do not describe this as "hosted" — it is local only.

## Plans

- Eventually move to the [[projects/homelab|Homelab]] (Oracle Cloud VM or equivalent)
- Possibly replace with a different meta-search engine if Whirlpool or another alternative fits better
- Could become a publicly accessible service once homelab infrastructure is stable
