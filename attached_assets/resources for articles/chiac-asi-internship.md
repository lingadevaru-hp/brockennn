ML-based metro train arrival and departure time predictor. Built from scratch — no scikit-learn.

- **Language:** Python 3.9+
- **Frontend:** Streamlit (interactive dashboard)
- **Database:** SQLite + MongoDB
- **ML Model:** Custom Perceptron-style Linear Regressor (gradient descent, L2 regularisation, early stopping)
- **Evaluation:** MAE, RMSE, R²
