# SI Coin (SIC) — Superintelligence Edge Currency

## Status
- **Research & presentation phase — Group 8, CHIAC ASI Internship.**
- Presentations delivered: 31 Mar 2026, 04 Apr 2026, 11 Apr 2026.

## What is SI Coin?
SI Coin (SIC) is a blockchain-based reward token tied to verified productive contribution within the Superintelligence Edge ecosystem at CHIAC ASI.

- Built on **Solana blockchain** (SPL token standard).
- **Max Supply:** 144,000 SIC (300 students × 12 weeks × 5 days × 8 hrs).
- Minted only by smart contract — no central authority.
- Unused SIC decays weekly (burn policy).
- Public ledger — all transactions visible.

## Energy Credit (EC) System
- 1 Energy Credit (EC) = 1 hour of verified productive contribution.
- Only validated output earns EC — time logged without output earns nothing.
- 70–80% of hours are convertible; unproductive time = 0 EC.
- Weekly cap: 40 SIC per student (maintains scarcity).

## How to Earn EC/SIC
| Category | Method | Notes |
|----------|--------|-------|
| Task Completion | Estimated hrs × Quality Factor (0.5x–1.5x) | Primary economy |
| Learning & Skills | Complete modules, share docs | Capped at 25% of total SIC |
| Helping Others | Help peer → receive 10–20% of their SIC | Collaboration reward |
| Integrity Bonus | Clean audit = 1.0x; dishonesty = 0.5x for 2 weeks | Trust layer |

## Reward Tiers
- **Tier 1 (Low SIC):** Digital badges, recognition, resume certification.
- **Tier 2 (Mid SIC):** Priority project selection, mentorship, recommendation letters.
- **Tier 3 (High SIC):** Stipend bonuses, fast-track hiring, leadership roles, equity participation.

## Token Architecture (4 Layers)
1. **Contribution Layer** — Student completes verified tasks → EC logged.
2. **Validation Layer** — Smart contract verifies quality → applies multiplier → approves mint.
3. **Token Minting Layer** — Approved EC → minted as SIC to student wallet.
4. **Reward & Redemption Layer** — SIC redeemable for badges, stipends, mentorship, equity access.

## Wallet Setup
- Token: SI Coin (SIC) | Standard: SPL (Solana)
- Wallet: Phantom Wallet
- Steps: Install Phantom → create Solana wallet → register wallet ID on SI platform → smart contract links wallet to profile → SIC auto-deposited after task validation.

## Global Architecture (V2 — April 4 Presentation)
- Expanded beyond CHIAC interns to global contributors.
- Any verified contributor worldwide can earn EC (students, teachers, developers, researchers, community members, organisations).
- 4 layers: Contribution → Validation (AI + peer review) → Token Issuance (blockchain mint) → Redemption (education, jobs, goods, real currency).
